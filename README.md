# GamePass Intelligence Dashboard

## About
An end-to-end data analytics project that treats a gaming platform's catalogue 
as a product portfolio. By analysing real-world game ratings, genres, player 
counts, pricing, and review trends, this project surfaces actionable business 
insights that a gaming company could use to make investment decisions.

## Key Questions
- Which genres retain players longest?
- Which genres are most reviewed — and what does that signal?
- Is there a correlation between price and rating?
- Can we predict whether a game will be a "hit" based on its features at launch?
